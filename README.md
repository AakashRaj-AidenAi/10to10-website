# 10to10 Adventures — Web Platform

Futuristic, scale-ready digital platform for 10to10 Adventures family entertainment venues.

See [`openspec/changes/revamp-10to10-futuristic-website/`](openspec/changes/revamp-10to10-futuristic-website/) for full proposal, research, design, and task list.

## Phase 1 MVP

- `apps/web` — Next.js 15 marketing + lead capture

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Stack (Phase 1)

Next.js 15 · React 19 · TypeScript · Tailwind v4 · shadcn/ui · Framer Motion · next-intl · Prisma · Zod · React Hook Form · Resend · PostHog · Sentry · Vercel
